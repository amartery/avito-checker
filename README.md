### Avito-requests
Сервис позволяет отслеживать количество объявлений по определнному поисковому запросу
## Запуск  
sudo docker-compose up  
## Тестирование  
http://127.0.0.1:8000/docs  
## Выполнено  
Данные сервиса хранятся во внешнем хранилище (MongoDB)
